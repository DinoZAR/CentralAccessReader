Highlight Algorithm
----------------------------------------------------------------------------------------

Here is where I will describe my algorithm for using the interactive word-by-word
feature using Javascript commands, as well as just general notes.



Overview:
---------

highlighting after that word will be done relatively. Therefore, some rules need to be
set as to how the highlighter goes from word to word (or section to section).
